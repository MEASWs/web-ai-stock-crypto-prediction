# Web AI Stock & Crypto Prediction

Artificial Intelligence web application for predicting **stock and cryptocurrency prices** using machine learning models.

## Technologies

* Python
* LSTM (Long Short-Term Memory)
* Machine Learning
* Streamlit
* Pandas / NumPy

## Features

* Data preprocessing pipeline
* LSTM deep learning model
* Ensemble prediction model
* Interactive web dashboard

## Dataset

Historical price data for stocks and cryptocurrencies including:

* AAPL
* AMZN
* BTC-USD
* ETH-USD
* GOOGL
* TSLA
* NVDA
* META

## How to Run

Install dependencies

```
pip install -r requirements.txt
```

Run the web application

```
streamlit run app.py
```
